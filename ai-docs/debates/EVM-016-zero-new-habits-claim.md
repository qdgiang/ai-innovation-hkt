# EVM-016 — Correct the “zero new habits” claim

> Priority: P2 · Status: `OPEN`

## Problem

Members avoid new weekly reporting forms, but leads still review proposals, exceptions, and
publication bundles. “Zero new habits” overpromises and hides a deliberate governance loop.

## Options

- **Option A — Keep the current phrase:** memorable but inaccurate.
- **Option B — Narrow the claim (`PROPOSED`):** “No new reporting destination for members; a lightweight
  exception-review loop for leads.”
- **Option C — Remove the claim:** safest but loses a useful explanation of ambient capture.

## Acceptance criteria

- Vision, pitch, demo script, and product copy use the same qualified claim.
- Copy distinguishes member reporting from lead governance.
- No flow claims to be habit-free when it requires explicit review.

---

## Resolution — 2026-07-18 (reviewed against design-v2 rev 13)

**RESOLVED — PARTIALLY ACCEPTED; qualified framing recorded, copy edits deferred.** The claim
actually got *stronger* since this was written: under settled #20 the bot never posts to
groups, so members' chat experience changes by exactly nothing — capture is ambient, approvals
ride gestures they already make (a reply, a 👍). The honest qualification is on the lead side,
and Option B's phrasing is adopted as the standard: **"no new reporting destination for
members; a lightweight dashboard review loop for leads"** (proposals, exceptions, digest).
Current copy is already close (VISION describes the write path and dashboard; the pitch pairs
"zero new habits" with the approval flow), so no edits ship today — the qualified phrasing
becomes binding at the next copy revision of VISION/pitch/demo script, per this file's
criteria.
